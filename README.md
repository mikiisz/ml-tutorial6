### Przygotowanie środowiska

```
    git clone https://github.com/mikiisz/ml-tutorial6.git && cd ml-tutorial6/

    #stworzenie srodowiska
    python -m venv ml6
    source ml6/bin/activate
    pip install -r requirements.txt

    #włączenie usługi jupyter lokalnie
    jupyter-notebook

    #deaktywacja venva
    deactivate

```
